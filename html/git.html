Git

  • git init - инициализация нового репозитория. Создает новый пустой репозиторий.
  • git clone - клонирует удаленный репозиторий на ваш локальный компьютер. Например, git clone https://github.com/username/repository.git.
  • git add - добавляет изменения в индекс для последующего коммита. Например, git add файл.txt.
  • git commit - сохраняет изменения в репозитории. Используйте git commit -m "Сообщение", чтобы добавить комментарий к коммиту.
  • git status - показывает текущее состояние репозитория, включая изменения, которые готовы для коммита.
  • git push - отправляет коммиты из вашего локального репозитория в удаленный. Например, git push origin main.
  • git pull - загружает и интегрирует изменения из удаленного репозитория в ваш локальный.
  • git branch - показывает список веток в репозитории. Используйте git branch имя_ветки, чтобы создать новую ветку.
  • git checkout - переключается на другую ветку или восстанавливает файлы. Например, git checkout main для переключения на ветку main.
  • git merge - объединяет изменения из одной ветки в другую. Например, git merge имя_ветки.
  • git log - показывает историю коммитов в текущем репозитории.
